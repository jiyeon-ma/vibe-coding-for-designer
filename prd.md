# PRD: Vibe Archive

> 디자이너를 위한 AI 기반 지능형 지식 외장하드

## 1. 목적 & 타겟

- **목적**: 디자이너가 수집한 영감(레퍼런스)과 기술 자산(개발/프롬프트)을 AI로 정제하여, 필요할 때 즉시 꺼내 쓸 수 있는 **지능형 지식 외장하드** 구축
- **타겟**: 코딩 문법보다 '바이브'와 '결과'에 집중하는 **AI 네이티브 디자이너**

## 2. 핵심 사용자 경험

| 키워드 | 설명 |
|---|---|
| **무지성 수집, 유지성 활용** | 외부에서 던진 링크를 AI가 알아서 분류·요약 |
| **디자이너의 제2의 뇌** | Git 명령어, 디자인 프롬프트를 체계적으로 관리 |
| **Inbox 기반 큐레이션** | '읽지 않은 것' 먼저 처리, '검증된 자산'만 남김 |

## 3. 주요 기능

### ① 통합 대시보드 & 검색 🌟 (메인 랜딩)
- **통합 검색바**: 최상단 배치 — 요약·태그·명령어 등 전체 검색
- **Smart Inbox**: 미확인 최신 레퍼런스를 카드 형태로 우선 노출
- **탭 네비게이션**: `전체` / `Visual Dictionary` / `Dev Terminal` / `Reference Hub` — 클릭 시 해당 섹션 앵커 이동

### ② Vibe Collector (AI 자동 수집)
- **외부 입력**: 카카오톡/MCP로 링크 전송 시 자동 아카이빙
- **AI 정제**: 링크 분석 → 한 줄 요약 + 카테고리/태그 자동 지정
- **액션**: Inbox에서 `보관(Archive)` → 해당 탭으로 이동 / `삭제`

### ③ Dev & Prompt Library (지식 자산화)
- **Visual Dictionary**: 감성 디자인 키워드 ↔ 구현 AI 프롬프트(Midjourney, three.js 등) 쌍 저장
- **Dev Terminal**: 디자이너 필수 Git 명령어, Claude Code 명령어, 개발 용어 정리
- **원클릭 복사**: 모든 코드·프롬프트에 복사 버튼 제공
